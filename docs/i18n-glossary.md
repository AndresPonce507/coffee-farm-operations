## Leave AS-IS (do not translate)
Brand/place: **Janson Coffee, Volcán, Chiriquí, Panamá**. Acronyms/units: **EUDR, QC, GPS, PHI,
REI, NDVI, NDRE, SAR, ATP, COGS, SCA, CVA, Brix, kg, ha, pH, °, %**. Code identifiers, testids,
classNames, URLs, lot codes (JC-101), proper names.

## Style
- Spanish (Panama), warm and plain — written for a working farm owner/crew, not a tech audience.
- Use `usted`-neutral imperatives for actions ("Guardar", "Registrar entrada").
- Keep `{var}` interpolation placeholders intact: `"{n} lotes"` → `"{n} lotes"`.
